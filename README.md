# 蓝眼泪模拟器

这是一个手机网页原型，当前版本改成了“透明容器中的发光液体”模型，而不是全屏随机粒子。

## 当前效果

- 透明瓶身边界
- 会随倾斜变化的液面
- 液体惯性导致的左右撞壁发光
- 底部撞击时的弧形荧光
- 液面边缘、瓶壁接触区、液体内部的蓝色余辉

## 使用方法

1. 解压 zip。
2. 用浏览器打开 `index.html`。
3. 手机端点击“启用运动感应”。
4. 轻微倾斜手机，观察液面倾斜和边缘亮度变化。
5. 快速晃动手机，观察液体撞击左右瓶壁和底部时的蓝色荧光。
6. 电脑端或未授权传感器时，可直接拖动屏幕搅动液体。

## 模型说明

核心状态大致分成：

- `fillLevel`
- `horizontalOffset`
- `surfaceTilt`
- `waveEnergy`
- `wallGlowLeft`
- `wallGlowRight`
- `bottomGlow`
- `surfaceGlow`

输入逻辑是：

`加速度 / 拖动速度 -> 液体速度 -> 撞击瓶壁 / 底部 -> 局部蓝光 -> 缓慢衰减`

## 注意

- iPhone Safari 需要用户点击按钮授权运动传感器。
- 某些设备还会要求同时授权方向传感器。
- 某些浏览器要求 HTTPS 才允许读取运动传感器。
- 如果本地打开无法读取传感器，可以部署到 GitHub Pages、Netlify 或 Vercel。
